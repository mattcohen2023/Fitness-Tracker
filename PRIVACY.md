# Privacy Policy

This application is a personal-use project that integrates with the WHOOP API
to retrieve fitness data for the sole user who created and authorized it.

## Data Collection
This application accesses the following data from the authorized user's WHOOP
account: recovery scores, sleep metrics, daily physiological cycles, workout
data, and basic profile information.

## Data Storage
All retrieved data is stored locally in a private Notion database controlled
by the authorized user. No data is shared with third parties, sold, or used
for any purpose other than personal fitness tracking.

## Data Sharing
No data collected by this application is shared with any external party.

## Access
Only the authorized user has access to the data retrieved by this application.
